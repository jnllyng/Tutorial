Writing Prompts Examples

Example 1: Activity Generation

AInstructor

Make a warm-up activity for my class.

BChatGPT

This prompt is too vague because it gives no information about the class, the learning goal, the time required, or the student level, which makes the AI produce random or irrelevant activities.

AInstructor

Create a 10-minute warm-up activity for a first-year Communication class about active listening. Include a short scenario, two questions, and a pair discussion task.

BChatGPT

This prompt is effective because it provides the class context, identifies the student level, defines the time, and specifies the components needed for the activity so the AI can produce a clear and usable result.



Example 2: Writing Prompts / Lesson Warm-Ups

AInstructor

Give a writing prompt.

BChatGPT

This prompt lacks a topic, purpose, word length, and audience, so the AI cannot generate a meaningful task aligned with the goals of a Communication course.

AInstructor

Create a writing prompt for a 120–150 word paragraph where students practice persuasive writing about whether smartphones should be allowed in classrooms.

BChatGPT

This version works well because it clearly defines the writing purpose, length, and topic, allowing AI to generate an assignment that fits typical classroom expectations.



Example 3: Emails or Announcements

AInstructor

Write an email to students.

BChatGPT

This prompt provides no purpose, tone, or length, so the AI may produce an email that is irrelevant, overly long, or unsuitable for class communication.

AInstructor

Write a clear and friendly announcement reminding students about tomorrow’s in-class activity. Keep it under 80 words and use simple language.

BChatGPT

This prompt guides the AI by defining the message goal, tone, and length, resulting in a practical draft the instructor can easily refine.



Example 4: Brainstorming Class Examples

AInstructor

Give examples of bad communication.

BChatGPT

This prompt does not specify the context, number of examples, or intended audience, causing the AI to generate content that may be too random or unrelated to the lesson.

AInstructor

Provide three real-life workplace examples, each two or three sentences long, that illustrate poor communication for first-year Communication students.

BChatGPT

This prompt works because it defines the number of examples, the context, the length, and the student level, allowing the AI to produce relevant and teachable scenarios.



Example 5: Summarizing Student Writing

AInstructor

Summarize this essay.

BChatGPT

This prompt is unclear about the length, reading level, and purpose of the summary, and it may accidentally lead the AI to evaluate the writing instead of summarizing it.

AInstructor

Summarize the following student paragraph in two or three simple sentences for quick instructor review. Do not evaluate or grade it.

BChatGPT

This version is effective because it establishes the summary length, tone, and purpose, and it explicitly prevents AI from grading or giving inappropriate feedback.



Example 6: Creating Quiz Questions

AInstructor

Generate quiz for communication class.

BChatGPT

This prompt is too general because it does not define the topic, question type, or quantity, so the AI will create inconsistent or unusable quiz items.

AInstructor

Create five multiple-choice questions that check basic understanding of paraphrasing. Each question should include one correct answer and three distractors.

BChatGPT

This version is helpful because it clearly defines the topic, format, number of questions, and structure, enabling the AI to produce usable and consistent quiz items.
